Git

  • git add - Команда git add добавляет содержимое рабочего каталога в индекс (staging area) для последующего коммита, "перечислить все файлы в папке" или "вывести на экран список файлов в папке".
  • git checkout - Команда git checkout позволяет перемещаться между ветками, созданными командой git branch, что переводится как "сменить папку" или "перейти в другую папку".
  • git status - Команда git status показывает состояния файлов в рабочем каталоге и индексе: какие файлы изменены, но не добавлены в индекс; какие ожидают коммита в индексе, что переводится как "текущая рабочая директория". То есть команда pwd показывает, в какой сейчас папке мы находимся.
  • git branch - Команда git branch — главный инструмент для работы с ветвлением. С ее помощью можно добавлять новые ветки, перечислять и переименовывать существующие и удалять их..
  • git mv - расфшифровывается как"Move", что переводится как "переместить", "передвинуть". С помощью нее можно переместить, переименовать или удалить файл/
